1. 프로젝트 개요

  • 프로젝트명: sosig.shop — AI News & Security Platform
  • 목표: AI 기반 뉴스 자동화, 외신 큐레이션, 보안 분석(악성코드·CVE·KISA), 네트워크 트래픽 가시화, 공공 도메인 문학 번역 선집, 실시간 도시 데이터(지하철)를 한 플랫폼에 통합

  • 인프라: Web Server (Gabia, Flask/Nginx) ↔ GPU Server (NVIDIA DGX Spark GB10, Tailscale + Cloudflare Tunnel) — 모든 AI 워크로드·크롤러·cron 워커는 GB10에서 실행

3. 주요 기능

메인 페이지 카드 분류와 동일한 4·4·4 그룹 + 운영 페이지 별도.

🎧 콘텐츠

공통 인프라: GB10 cron이 매시 RSS·웹 크롤러를 실행해 AI/경제/사회 뉴스를 수집·필터링(중복·연예성 제거) → Claude Sonnet 4.6이 상현(진행자)·지민(분석가) 2인 대화체 대본으로 재구성 → Microsoft Edge TTS(ko-KR InJoon/SunHi/Hyunsu)로 자연스러운 오디오 생성.

🎙️ 팟캐스트 캘린더 (/podcast)

날짜별·주제별 일일 AI 뉴스 브리핑. 시간대별 그룹과 검색·필터 제공.

🎧 시간대별 모음 (/briefing)

새벽/아침/낮/저녁/밤 5개 시간대 + 일별 전체 모음. ffmpeg concat으로 자동 생성하며 기사 사이 1.5초 무음으로 트랙 구분.

📹 AI 뉴스 영상 (/news/videos)

실제 기사 본문을 기반으로 AI가 영상 클립과 한국어 내레이션을 자동 생성. 날짜별 영상 모음.

🗞️ 외신이 본 한국 (/foreign)

BBC News(Business / Asia / World / Technology 4피드) + Nikkei Asia RSS를 4시간마다 수집, Claude Haiku가 한국 관련성(경제·정치·외교·안보) 판별 + 네이버 검색으로 국내 매체 미커버 여부 확인. 통과한 기사만 Claude Sonnet 대화체 대본(TTS용) + 한국어 번역본(UI 표시용)을 함께 생성하여 팟캐스트로 발행.

🛡️ 보안

🇰🇷 KISA 보안 권고 (/kisa)

한국인터넷진흥원 보호나라(boho.or.kr) 보안공지·취약점·공지를 매시 스크래핑하여 Claude Haiku로 한국어 요약·우선순위·CVE/제품 추출. priority≥4 권고는 자동으로 12~16줄 보안 안내 팟캐스트로 발행(인트로 + chime + 대화). 사이버 위기 단계도 함께 표시.

🛡️ CVE 취약점 모니터링 (/vuln)

NVD에서 주요 소프트웨어 CVE를 2시간마다 수집, Claude Haiku로 한국어 요약·심각도·악용 가능성 평가. OpenVAS 자기 인프라 스캔 결과와 cross-link하여 노출된 자산만 강조.

🔬 악성코드 분석기 (/malware)

PE(Windows) 및 ELF(Linux) 실행 파일 업로드 → 다단계 정적 분석 + AI 분석.

  • 바이너리 파싱: LIEF로 PE/ELF 헤더·임포트 테이블·섹션 정보 추출
  • API 분석: 12개 카테고리의 위험 API 탐지 (프로세스 인젝션, 안티디버깅, 키로깅, 권한 상승 등)
  • 엔트로피·문자열 분석: 패킹/난독화 탐지, 의심 문자열 위험도별 색상 분류
  • YARA 시그니처: 1,200+ 커뮤니티 룰 + 커스텀 룰 (매일 자동 업데이트)
  • 디스어셈블리 + AI 디컴파일: Capstone(x86/x64/ARM/AArch64) → Claude Sonnet이 C 의사코드 변환, 의심 코드 자동 하이라이트
  • AI 판단: Claude Sonnet이 MITRE ATT&CK 기반 종합 위협 분석 + 한국어 번역, SSE 실시간 스트리밍

🏢 인프라 스캔 (/vuln/infra)

OpenVAS로 자기 인프라(웹·DB·SSH 등) 정기 풀스캔. 발견된 취약점과 호스트별 위험도를 한눈에 보고 CVE 페이지와 cross-link.

🛠️ 도구·자료

🌐 네트워크 경로 추적 (/route)

TCP/UDP traceroute (scapy 기반, 3-way 폴백) 결과를 방사형 hop 다이어그램으로 시각화 — 내 LAN 중심에서 동심원(TTL 단계)으로 외부 경로를 펼친다.

🚇 지하철 실시간 (/subway)

서울 지하철 노선의 실시간 열차 위치(서울교통공사·국토교통부 OpenAPI)를 노선도 위에 시각화. 신분당선부터 시작하여 점진 확장 중.

🔒 네트워크 트래픽 분석 (/network, 본인 전용)

TL-SG105E port mirror로 LAN 전체 트래픽을 GB10의 USB-LAN으로 미러링 → Zeek + Suricata 24시간 캡처 → vLLM Gemma 4 26B-A4B로 1차 triage + 의심 호스트 deep dive → 시간대별/호스트별 통계와 지구본 시각화. 본인 전용 텔레그램 OTP 인증.

📚 AI 번역 선집 (/book)

퍼블릭 도메인 문학을 vLLM Gemma 4 26B-A4B로 2-pass 번역하여 책 형태 PDF로 발행. 니체 『위버멘쉬』, 장자 내편, 하워드 파일 『아서왕』, 잰그윌 『보우의 미스터리』, 에드거 앨런 포 단편·시 등 총 9권. A5 판형, Noto Serif CJK KR 본문.

⚙️ 운영·메타 (메인 푸터에서 진입)

📅 캘린더 UI

날짜별·토픽별로 팟캐스트와 영상을 탐색. 전체 검색·필터 제공.

📋 방명록 (/broadcast)

방문자가 소감과 메시지를 남길 수 있는 페이지.

⚙️ 운영 대시보드 (/status)

플랫폼 각 서비스의 카드형 실시간 상태: 영상·팟캐스트·KISA·vuln·악성코드·OpenClaw·GB10 시스템 등.

4. 기술 스택

Backend

  • Flask + Gunicorn: Blueprint 기반 모듈형 웹 서버
  • Nginx: 리버스 프록시, SSL 터미네이션, 정적 파일 캐싱
  • MariaDB: 뉴스 메타데이터, 사용자 데이터 저장

AI / ML

  • vLLM Gemma 4 26B-A4B FP8 + MTP drafter (GB10 로컬, 메인 LLM): 뉴스/외신/KISA 대본, 책 번역, 네트워크 트래픽 분석, 악성코드 보조 분석 — Ollama 56s → 7s (8배 가속), Claude Sonnet 비등 품질, 비용 0
  • Ollama qwen3.6:35b (로컬): 악성코드 분석기(특화 추론), OpenClaw cron 트리거
  • qwen2.5-coder (로컬): 디컴파일·디스어셈블리 코드 해석
  • Claude CLI (Opus 4.7 / Sonnet 4.6 / Haiku 4.5): 인터랙티브 개발 세션, OpenClaw 잔존 cron(claude update·gx10 update·OpenClaw update-check)
  • Supertonic TTS (on-device 44.1kHz): 한국어 음성 합성 (M1/F1/M2 페르소나)

보안 분석

  • LIEF: PE/ELF 바이너리 파싱
  • Capstone: 멀티 아키텍처 디스어셈블리 엔진 (x86/x64/ARM/AArch64)
  • YARA: 시그니처 기반 악성코드 탐지 (Signature-Base, YARA-Rules 커뮤니티)
  • Suricata 8 + ET Open: IDS 룰 50,000+ 으로 LAN mirror 트래픽 실시간 탐지
  • Zeek (Docker): 프로토콜 분석 — conn/dns/ssl/http/files 로그를 일별 회전·압축
  • OpenVAS: 자기 인프라 풀스캔 결과를 vuln 페이지와 cross-link
  • 커스텀 위험 API DB: 12개 카테고리, 60+ API 매핑

인프라

  • NVIDIA DGX Spark GB10: ARM64 + Grace Blackwell. ollama·악성코드 API·뉴스/KISA 워커·네트워크 분석을 모두 호스트
  • Tailscale: Gabia ↔ GB10 mesh VPN (MySQL/HTTP 직통, 외부 노출 0)
  • Cloudflare Tunnel: malware-api.sosig.shop만 외부 노출 (Bearer Token + WAF)
  • Telegram Bots: @sosig_taeng_bot (OTP/알림) + @sosig_video_bot (영상 등록)
  • TL-SG105E port mirror: LAN port 1 → port 5 미러 (Ingress+Egress) → GB10 USB-LAN 캡처
  • Cron 자동화: 뉴스/KISA/vuln/네트워크 분석/팟캐스트 발행 (KST 기준 매시·매 2시간·매일)

  • 실시간 스트리밍: SSE(Server-Sent Events)로 AI 분석, 번역, 디컴파일 결과를 실시간 스트리밍, Nginx proxy_buffering 최적화
  • YARA 룰 자동화: Signature-Base, YARA-Rules 등 커뮤니티 룰셋을 일일 자동 다운로드, 유효성 검증 후 적용
  • 로컬 실행 최적화: Windows Task Scheduler 기반 자동화로 Cloud Run의 SFTP 제약 및 네트워크 제한 문제 해결
  • 프롬프트 엔지니어링: 단순 요약을 넘어선 '페르소나(상현/지민)' 기반의 몰입감 있는 대본 생성
  • API Rate Limit 보호: Gemini API 호출 후 7초 대기 및 실패 시 30초 재시도로 429 오류 방지
