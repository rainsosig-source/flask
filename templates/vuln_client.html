📥 클라이언트 다운로드

리눅스 서버에서 아래 명령어로 클라이언트를 다운로드하고 실행하세요.

📥 vuln_checker.py 다운로드 🔗 GitHub

1단계: 다운로드

# wget으로 다운로드
wget https://raw.githubusercontent.com/rainsosig-source/CVE_vulnerability/main/client/vuln_checker.py

# 또는 curl로 다운로드
curl -o vuln_checker.py https://raw.githubusercontent.com/rainsosig-source/CVE_vulnerability/main/client/vuln_checker.py

2단계: 의존성 설치

pip3 install requests

3단계: 실행

# 기본 실행 (탐지 → 스캔 → 검증)
python3 vuln_checker.py

# JSON 출력 (자동화용)
python3 vuln_checker.py --json

# 검증 코드 실행 안함 (빠른 스캔)
python3 vuln_checker.py --no-verify

1️⃣ NVD API에서 CVE 수집

🌐

NIST NVD API 연동

미국 국립표준기술연구소(NIST)의 National Vulnerability Database에서 실시간 CVE 정보 수집

🔍

키워드 기반 검색

Apache, Nginx, MySQL 등 17종 소프트웨어별 키워드로 관련 CVE 필터링

# NVD API 호출 예시
GET https://services.nvd.nist.gov/rest/json/cves/2.0
?keywordSearch=nginx&pubStartDate=2024-01-01

📡 API 엔드포인트

고급 사용자를 위한 API 직접 호출:

Method Endpoint 설명
GET /api/client/software 모니터링 대상 소프트웨어 목록
POST /api/client/scan 취약점 스캔 요청
GET /api/client/verify/{cve_id} 검증 코드 다운로드
GET /api/client/download 클라이언트 스크립트 다운로드

API 예제

# 소프트웨어 목록 조회
curl https://sosig.shop/api/client/software

# 취약점 스캔
curl -X POST https://sosig.shop/api/client/scan \
-H "Content-Type: application/json" \
-d '{"software": [{"name": "nginx", "version": "1.18.0"}]}'
