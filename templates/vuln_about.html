🛡️ 취약점 모니터링

CVE 뉴스 피드 + 내 인프라 실측 진단, 두 관점을 교차하는 개인 SOC.

개요

이 사이트는 세상에 나오는 전체 CVE를 모니터링하는 뉴스 레이어와, 실제 내 서버/네트워크에 존재하는 취약점을 진단하는 스캔 레이어를 함께 제공합니다. 두 데이터는 CVE ID로 교차돼, "방금 뉴스에 나온 이 취약점이 우리 인프라에도 있다"는 통합 관점을 만듭니다.

두 개의 레이어

1) CVE 뉴스 (/vuln)

NVD(국가 취약점 DB)를 2시간 주기로 크롤링해 세상의 모든 신규 취약점을 수집합니다. 각 CVE에 대해 LLM이 한국어 요약·영향·대응 방안을 자동 생성합니다.

2) 내 인프라 진단 (/vuln/infra)

OpenVAS(Greenbone Community Edition)가 실제 서버·네트워크를 주기적으로 스캔해 "이 서버의 이 포트·이 패키지에 이 CVE가 존재한다"는 구체적 findings을 생성합니다. 공인 IP/사설 IP는 public-host-N / lan-host-N 라벨로 마스킹됩니다.

통합: 뉴스 × 인프라 교차

DB 뷰 v_my_infrastructure_cves가 NVD CVE와 OpenVAS findings를 CVE ID로 교차해 "내 자산에 실존하는 CVE" 만 추려 보여줍니다.

공개 정책

  • 사설 IP(192.168.x.x)는 gateway / scanner / lan-host-N 로 마스킹
  • 공인 IP는 public-host-N 으로 마스킹 (Cloudflare 뒤 오리진 보호)
  • 개별 호스트 상세 URL은 IP가 아닌 slug 기반. IP 직접 조회 불가
  • MySQL 3306은 인터넷 미노출, Tailscale 전용
